Fonts

The top three heading levels use the typeface Roboto Black. The fourth heading level and the running text use the typeface Oxygen. These are both web fonts, freely available so there are no copyright encumbrances. We are only using the black version of Roboto, since it's only used as a heading. Oxygen is available in regular, italic, bold, and bold-italic. Here we are mostly using the regular. Because we have styled <i> and <b>, incidental italics and bolds can be coded in the usual ways, without referencing the specific OxygenItalic or OxygenBold faces. Even bold italics should work.

For monospaced type (e.g., to indicate sample code), we use Oxygen Mono.

Logo

The KBase logo should always be used with a white or very light gray background, and plenty of "breathing room" around it. The distance between the logo and its nearest item should always be greater than the distance between elements within the logo itself. Be sure you are using the version that has lowercase "ase" in KBase. All the elements of the logo must be present. Do not alter colors in the logo.

Colors

KBase web pages should have a predominantly white or very light gray background. Accent colors should be used sparingly, to keep the feel light and clean.
